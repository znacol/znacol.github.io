PROJECTS

Smart Lots Project

In collaboration with my school's ACM chapter, I have been working on a project called Smart Lots. The goal is to design a system to monitor the parking lots on campus and offer real time parking information. Students can go to a website and see what lots still have spaces available, preventing them from driving all over campus. The system currently consists of an Arduino with a sensor that detects changes in the magnetic field. The Arduino sends the data to a Raspberry Pi that then parses the data and sends to a server, updating the web page.
